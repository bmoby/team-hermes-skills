# Client Idea Analyst — Bots

Use this skill when you want to explore bot-related ideas from a validated client profile.

## Goal

Generate as many useful bot ideas as possible for customer interactions or internal handling.

## Scope

Typical themes:
- Telegram bot
- WhatsApp bot
- Discord bot if relevant
- conversational intake
- booking support
- FAQs
- request qualification
- support automation
- client follow-up via chat

## Rules

- Work from the validated client profile.
- Keep the output lightweight.
- One short sentence per idea is enough.
- Do not rank the ideas unless asked.
- Do not reject ideas too early.
- Do not write technical specs.
