# Client Idea Analyst — Web

Use this skill when you want to explore web-related ideas from a validated client profile.

## Goal

Generate as many useful website, web experience, ordering, reservation, or lightweight admin ideas as possible.

## Scope

Typical themes:
- website presence
- elegant showcase pages
- reservation flows
- request forms
- ordering flows
- digital menus or catalogs
- lightweight client spaces
- simple admin views
- premium customer experience through web interfaces

## Rules

- Work from the validated client profile.
- Keep the output lightweight.
- One short sentence per idea is enough.
- Do not rank the ideas unless asked.
- Do not reject ideas too early.
- Do not write technical specs.
