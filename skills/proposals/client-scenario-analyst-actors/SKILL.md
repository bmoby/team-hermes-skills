# Client Scenario Analyst — Actors

Use this skill to generate likely actors and user profiles before writing scenarios.

## Goal

Create a broad actor catalog from the upstream artifacts.
Include both normal and difficult profiles, external and internal.

## Scope

Typical actors may include:
- customers
- premium customers
- rushed customers
- confused customers
- returning customers
- dishonest or abusive customers
- elderly users
- parents / families if relevant
- owner
- manager
- staff
- operators
- partners / subcontractors if relevant

## Rules

- Work from the upstream artifacts.
- Keep profiles short but concrete.
- Include realistic variety.
- Include enough profiles to support a large scenario run, not just a handful.
- Do not write full stories here.
